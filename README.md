# AbutionGraph（时序图数据库）
AbationGraph® is a time-series knowledge graph database for real-time OLAP。  
AbationGraph - 一款面向时序多维关联数据分析（GraphOLAM）的图谱数据库系统，它提供大规模时序数据流端到端的自动聚合计算并更新历史数据，数据实时接入后立即查看到关联计算产生的结果。
